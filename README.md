# cochumo-ui

This is a sample UI component

## installation

```bash
$ npm install cochumo-ui @emotion/react @emotion/styled
```

or

```bash
$ yarn add cochumo-ui @emotion/react @emotion/styled
```

## example

```jsx
import { Text } from 'cochumo-ui'

<Text text="hello, world" />
```

## develop component

```bash
$ yarn install
$ yarn run storybook
```
